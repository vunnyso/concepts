## Difference between .go and .pb.go file

| Feature        | .go file              | .pb.go file                         |
| -------------- | --------------------- | ----------------------------------- |
| Who writes it? | Developer             | Generated by protoc                 |
| Purpose        | Application logic     | Data structure & serialization code |
| Based on       | Custom implementation | .proto file definitions             |
| Modifiable?    | Yes                   | No (should not be edited manually)  |
