### Build and Test
1. Install the rust on the system

2. Check rust compiler and cargo version
   ```sh
   rustc --version
   cargo --version
   ```

3. Create a new project
   ```sh
   cargo new test_project
   cd test_project
   ```

4. Compile and test
   ```sh
   cargo build
   cargo run
   ```
